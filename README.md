# Marketing-for-Nura-Labs-Case-Study

This repository contains a case study of a marketing campaign I executed for a **crypto project** called Nura Labs. The campaign focused on promoting the project, building brand awareness, and leveraging digital tools to enhance marketing efforts.

- **Results & Insights**: Key outcomes from the campaign and what I learned.
